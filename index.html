FORGE User Guide

What FORGE is

FORGE is a zero-setup productivity app skeleton builder. It runs as one local HTML file or from GitHub Pages. Projects stay local only.

Basic workflow

  1. Pick a blueprint or start blank.
  2. Add components from the left panel.
  3. Select, drag, resize, and edit in the inspector.
  4. Create pages and connect buttons with actions.
  5. Use Preview mode to test navigation.
  6. Export .forge.json for backup and HTML/ZIP for demos.

Local-only rules

No login. No server. No cloud save. No GitHub save. GitHub Pages may host only the tool.

Important shortcuts

Ctrl+C/V copy paste, Ctrl+Z/Y undo redo, Delete removes, Ctrl+A selects all, Shift+Arrow moves 10px, Alt+Arrow moves 1px, Ctrl+Scroll zooms.
